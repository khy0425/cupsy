# Cupsy 앱 빠른 시작 가이드

## 소개

Cupsy는 사용자의 감정과 상황을 음료로 시각화해주는 앱입니다. 매일 하루에 한 번, 자신의 감정과 상황을 선택하면 그에 맞는 고유한 감정 음료를 생성하고, 이를 저장하거나 공유할 수 있습니다.

## 앱 설치 및 실행

1. 개발 환경 요구사항:
   - Flutter 3.10.0 이상
   - Dart 3.0.0 이상
   - Android Studio 또는 VS Code

2. 프로젝트 설치:
   ```
   git clone https://github.com/your-username/cupsy.git
   cd cupsy
   flutter pub get
   ```

3. 앱 실행:
   ```
   flutter run
   ```

## 앱 구조

Cupsy 앱은 다음과 같은 구조로 구성되어 있습니다:

- `lib/main.dart`: 앱 진입점
- `lib/models/`: 데이터 모델 (Emotion, Situation, EmotionCup 등)
- `lib/providers/`: 상태 관리 (Riverpod 기반)
- `lib/screens/`: 앱 화면 구현
- `lib/utils/`: 유틸리티 함수 및 클래스
- `lib/widgets/`: 재사용 가능한 위젯
- `lib/theme/`: 앱 테마 및 스타일 정의

## 주요 기능

1. **감정 선택**: 다양한 감정 중 하나를 선택할 수 있습니다.
2. **상황 선택**: 감정을 느낀 상황을 선택할 수 있습니다.
3. **감정 음료 생성**: 선택한 감정과 상황에 따라 고유한 음료가 시각화됩니다.
4. **이미지 저장**: 생성된 감정 음료를 갤러리에 저장할 수 있습니다.
5. **공유 기능**: SNS 등을 통해 감정 음료를 공유할 수 있습니다.
6. **일일 생성 제한**: 하루에 한 번만 음료를 생성할 수 있습니다.

1. **AdMob 통합**:
   - Google AdMob 계정 및 앱 등록

3. **감정 음료 저장소**:
   - 과거 생성한 감정 음료 목록 표시
   - 필터링 기능 (날짜별, 감정별, 상황별)
   - 정렬 옵션 (최신순, 감정별 등)

4. **테마 및 설정**:
   - 다크 모드 지원
   - 사용자 지정 색상 테마
   - 앱 언어 설정 (다국어 지원)
   - 알림 설정

5. **AI 통합 (선택사항)**:
   - 감정 분석을 위한 AI API 연동
   - 사용자 입력 텍스트 기반 감정 자동 분석
   - 생성형 AI를 이용한 맞춤 설명 생성
